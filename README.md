Baixing

百姓网笔试测试

所有css写在html文件中，方便查看。

1. 第一题 文件：index.html
   * 样式reset参考AliceUI
   * HTML5
   * 语义化结构
   * 兼容所有主流浏览器（IE6+ / Chrome / Firefox / Safari / Opera）
   * 无空标签
2. 第二题 文件：html5-login.html
   * HTML5 注册表单
   * 语义化结构
   * 跨浏览器兼容的Placeholder属性
   * 加入表单验证优化
3. 第三题 文件：event.js 测试用例：demo.html
   * 兼容IE6.0+,Safari,Chrome,Firefox以及W3C模型
   * 自定义事件注册和注销
   * 阻止事件捕获和事件冒泡
